# i18n

组件库内置组件文案(占位符、ARIA 标签、按钮文字)通过单一 context provider 进行本地化。

## 默认

无 provider 时所有组件使用**俄语**(`ru`)。`Очистить`、`Поиск...`、`Готово` 等文案来自这里。

## 切换为英文

用 `UidLocaleProvider` 包裹应用并传入内置 `en`：

```vue
<script setup lang="ts">
import { UidLocaleProvider, en } from '@dskripchenko/ui'
</script>

<template>
  <UidLocaleProvider :locale="en">
    <App />
  </UidLocaleProvider>
</template>
```

就这样。所有内置文案(清除按钮、搜索占位符、Tour 导航等)都会变为英文。

## 内置 locale

```ts
import { ru, en } from '@dskripchenko/ui'
```

- `ru` — 默认。俄文。
- `en` — 英文。

两者均实现完整的 `UidLocale` 接口。

## 部分覆盖

无需提供完整 locale。Provider 会把你的覆盖深度合并到默认 `ru` 之上：

```vue
<UidLocaleProvider :locale="{ tour: { next: '前进', prev: '后退' } }">
  <App />
</UidLocaleProvider>
```

未指定的键保留 `ru` 的值。英文同理：

```vue
<script setup lang="ts">
import { en } from '@dskripchenko/ui'
const locale = { ...en, tour: { ...en.tour, finish: '搞定！' } }
</script>

<template>
  <UidLocaleProvider :locale="locale">
    <App />
  </UidLocaleProvider>
</template>
```

## 自定义 locale

通过实现 `UidLocale` 提供完整 locale：

```ts
import type { UidLocale } from '@dskripchenko/ui'

export const zhCN: UidLocale = {
  common: {
    clear: '清除',
    close: '关闭',
    search: '搜索...',
    confirm: '确认',
    cancel: '取消',
    loading: '加载中...',
    noResults: '无结果',
  },
  copy: { copy: '复制', copied: '已复制' },
  select: { placeholder: '请选择...', noResults: '无结果' },
  combobox: {
    placeholder: '开始输入...',
    noResults: '无结果',
    create: (label) => `+ 创建"${label}"`,
  },
  datePicker: {
    placeholder: '选择日期',
    months: ['一月', '二月', '三月', '四月', '五月', '六月',
             '七月', '八月', '九月', '十月', '十一月', '十二月'],
    weekdaysShort: ['一', '二', '三', '四', '五', '六', '日'],
    prevMonth: '上一月',
    nextMonth: '下一月',
  },
  // … 完整结构见 UidLocale 类型
}
```

像内置那样使用：

```vue
<UidLocaleProvider :locale="zhCN">
  <App />
</UidLocaleProvider>
```

## 在自己代码中读取 locale

如果你基于组件库构建组件并希望使用当前 locale：

```vue
<script setup lang="ts">
import { useLocale } from '@dskripchenko/ui'

const locale = useLocale()
</script>

<template>
  <span>{{ locale.common.clear }}</span>
</template>
```

`useLocale()` 返回 `ComputedRef<UidLocale>`，模板中自动解包。

## 编程式 provide

如果需要编程式提供 locale(例如下拉框切换语言)：

```vue
<script setup lang="ts">
import { ref, computed } from 'vue'
import { provideLocale, ru, en } from '@dskripchenko/ui'

const lang = ref<'ru' | 'en'>('en')
provideLocale(computed(() => (lang.value === 'ru' ? ru : en)))
</script>

<template>
  <select v-model="lang">
    <option value="ru">俄文</option>
    <option value="en">英文</option>
  </select>
  <App />
</template>
```

## 覆盖范围

所有组件文案(占位符、ARIA 标签、按钮文字)都通过 provider 本地化。已覆盖的组件：

Select、Combobox、DatePicker、DateRangePicker、TimePicker、TreeSelect、NumberInput、TagsInput、FileUpload、Mention、BackTop、Tour、TreeView、Pagination、Modal、Drawer、Toast、Alert、Tag、Code、DescriptionList、ColorPicker、Cascader、Calendar 等。

## Prop 级覆盖

prop 级定制依然优先于 locale。例如 `UidSelect`：

```vue
<UidSelect placeholder="三选一..." :options="options" />
```

`placeholder` prop 优先于 `locale.select.placeholder`。仅当 prop 缺省时 locale 才生效。

## Locale 形态

```ts
interface UidLocale {
  common: {
    clear: string; close: string; search: string
    confirm: string; cancel: string; loading: string; noResults: string
  }
  copy: { copy: string; copied: string }
  select: { placeholder: string; noResults: string }
  combobox: { placeholder: string; noResults: string; create: (label: string) => string }
  datePicker: {
    placeholder: string
    months: readonly string[]
    weekdaysShort: readonly string[]
    prevMonth: string; nextMonth: string
  }
  dateRangePicker: { placeholder: string; presetLast: (days: number) => string }
  timePicker: { placeholder: string; now: string; confirm: string }
  colorPicker: { placeholder: string }
  numberInput: { increment: (step: number | string) => string; decrement: (step: number | string) => string }
  tagsInput: { remove: (label: string) => string }
  treeView: { expand: string; collapse: string }
  treeSelect: { placeholder: string; removeTag: (label: string) => string }
  fileUpload: { primaryText: string; secondaryText: string; remove: (name: string) => string }
  backTop: { label: string }
  tour: { next: string; prev: string; finish: string; skip: string }
  rating: { label: (current: number, max: number) => string }
  mention: { noResults: string }
  modal: { close: string }
  drawer: { close: string }
  toast: { close: string }
  pagination: {
    first: string; prev: string; next: string; last: string
    page: (n: number) => string
    pageSize: (n: number) => string
  }
}
```

部分覆盖请使用 `UidPartialLocale` — 每个嵌套键都变为可选。
